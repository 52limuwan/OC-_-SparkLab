# 课程学习功能

## 功能概述

学生在课程中心注册课程后，可以进入课程学习页面，查看课程下的所有实验，并进入具体的实验进行学习。

## 功能流程

### 1. 课程中心（/explore）
- 显示所有已发布的课程
- 未注册课程显示"立即注册"按钮
- 已注册课程显示"进入学习"按钮
- 点击"进入学习"跳转到课程详情页

### 2. 课程详情页（/courses/[id]）
- 显示课程基本信息（标题、描述、难度、时长、进度）
- 显示课程下的所有实验列表
- 每个实验显示：
  - 序号
  - 标题和描述
  - 难度等级
  - 时间限制
  - 分数
  - "进入学习"按钮
- 点击"进入学习"跳转到实验页面

### 3. 实验学习页（/lab/[id]）
- 左侧：实验内容
  - 实验标题和描述
  - Markdown 格式的实验内容
  - 实验任务列表
  - 返回大厅和提交实验按钮
  
- 右侧：实例操作区
  - 三个标签页：VNC、SSH、RDP
  - 容器控制按钮：启动容器、停止容器、清空输出
  - VNC 标签：图形界面连接（开发中）
  - SSH 标签：终端命令行界面
  - RDP 标签：远程桌面连接（开发中）

## 技术实现

### 前端路由
- `/explore` - 课程中心
- `/courses/[id]` - 课程详情页（新增）
- `/lab/[id]` - 实验学习页

### API 接口
- `GET /courses` - 获取所有课程
- `GET /courses/:id` - 获取课程详情
- `POST /courses/:id/enroll` - 注册课程
- `GET /labs/course/:courseId` - 获取课程下的所有实验
- `GET /labs/:id` - 获取实验详情
- `POST /containers` - 创建容器
- `POST /containers/:id/start` - 启动容器
- `POST /containers/:id/stop` - 停止容器
- `POST /containers/:id/exec` - 执行命令

### 数据模型
- Course - 课程
- Lab - 实验
- Enrollment - 课程注册记录
- Container - 容器实例

## 用户体验

1. 学生在课程中心浏览课程
2. 点击"立即注册"注册课程，按钮变为"进入学习"
3. 点击"进入学习"进入课程详情页
4. 在课程详情页查看所有实验
5. 点击实验的"进入学习"按钮进入实验页面
6. 在实验页面左侧阅读实验内容，右侧操作容器
7. 可以选择 VNC、SSH 或 RDP 方式连接容器
8. 完成实验后点击"提交实验"
