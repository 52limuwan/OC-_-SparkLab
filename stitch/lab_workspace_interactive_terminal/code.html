Lab 01: Nginx Configuration

Overview

In this lab, you will deploy a high-performance Nginx container and configure a custom static landing page. You'll learn how to map volumes and expose specific ports to the host machine.

"Containerization is not just about isolation, but about reproducible environments."

Steps

1. Pull the official Nginx image docker pull nginx:latest
2. Create a local content directory

Create a directory named html in your home directory.

3. Run the container with Volume Mapping docker run -d -p 8080:80 --name my-web -v ~/html:/usr/share/nginx/html nginx
4. Verify accessibility

Access the service via localhost:8080 and confirm your custom index.html is active.
