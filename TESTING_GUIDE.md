# 课程学习功能测试指南

## 前置准备

### 1. 启动后端服务
```bash
cd backend
npm install
npm run start:dev
```

### 2. 初始化数据库
```bash
cd backend
npx prisma migrate dev
npx prisma db seed
```

### 3. 启动前端服务
```bash
cd frontend
npm install
npm run dev
```

## 测试流程

### 1. 登录系统
1. 访问 http://localhost:3000
2. 点击"登录"按钮
3. 使用学生账号登录

### 2. 浏览课程中心
1. 登录后自动跳转到大厅
2. 点击侧边栏的"课程中心"或访问 `/explore`
3. 查看所有可用课程：
   - Linux 基础入门（入门级）
   - Docker 容器技术（进阶级）

### 3. 注册课程
1. 在课程卡片上点击"立即注册"按钮
2. 注册成功后，按钮变为"进入学习"
3. 课程状态已保存到数据库

### 4. 进入课程详情
1. 点击"进入学习"按钮
2. 跳转到课程详情页 `/courses/[id]`
3. 查看课程信息：
   - 课程标题和描述
   - 难度等级
   - 预计时长
   - 学习进度
4. 查看实验列表：
   - 实验序号
   - 实验标题和描述
   - 难度、时长、分数

### 5. 进入实验学习
1. 点击实验的"进入学习"按钮
2. 跳转到实验页面 `/lab/[id]`
3. 左侧显示实验内容：
   - 实验标题和描述
   - Markdown 格式的实验指导
   - 实验步骤（如果有）
4. 右侧显示实例操作区：
   - VNC、SSH、RDP 三个标签
   - 容器控制按钮

### 6. 操作容器
1. 点击"启动容器"按钮
2. 等待容器创建完成
3. 切换到 SSH 标签
4. 在终端输入命令测试：
   ```bash
   pwd
   ls -la
   echo "Hello Spark Lab"
   ```
5. 查看命令执行结果

### 7. 切换连接方式
1. 点击 VNC 标签查看图形界面（开发中）
2. 点击 RDP 标签查看远程桌面（开发中）
3. 点击 SSH 标签返回终端

### 8. 提交实验
1. 完成实验任务后
2. 点击"提交实验"按钮
3. 系统记录提交信息

### 9. 返回课程
1. 点击"返回大厅"按钮
2. 或通过侧边栏导航返回课程中心
3. 继续学习其他实验

## 常见问题

### 1. 容器启动失败
- 检查 Docker 服务是否运行
- 检查后端日志查看错误信息
- 确认 Docker 镜像是否存在

### 2. 命令执行无响应
- 检查容器是否正常运行
- 查看浏览器控制台错误
- 检查后端 API 是否正常

### 3. 页面跳转失败
- 检查路由配置
- 确认用户已登录
- 查看浏览器控制台错误
